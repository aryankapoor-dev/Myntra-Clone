## 📖 About

This project recreates the **Myntra Home Page** using **React** and **Parcel**.

Built as my **first React project**, it focuses on learning component-based architecture, JSX, reusable UI, and responsive frontend development while recreating one of India's most popular e-commerce websites.

---

## ✨ Features

* 🛍️ Responsive Myntra landing page
* ⚛️ Component-based React architecture
* 🎨 Clean and modern UI
* 📱 Responsive layout
* 🚀 Fast development with Parcel
* ♻️ Reusable components

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/aryankapoor-dev/Myntra-Clone.git
```

Go to the project directory

```bash
cd Myntra-Clone
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm start
```
